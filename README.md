\# Pharmacy Management System



A desktop-based \*\*Pharmacy Management System built using Python and Tkinter\*\*.

This system helps manage medicines, admin authentication, and basic pharmacy operations in a simple graphical interface.



\## Features



\- Admin Login System

\- Add / Delete Medicines

\- Search Medicine

\- Change Password

\- Change Admin Username

\- Clean Dashboard UI

\- Activity Logging



\## Technologies Used



\- Python

\- Tkinter (GUI)

\- SQLite Database



\## Installation



1\. Clone the repository



git clone https://github.com/Imrankhanjadoon/pharmacy-management-system.git



2\. Navigate to project folder



cd pharmacy-management-system



3\. Run the program



python pharmacy.py
